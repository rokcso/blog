我的 macOS 初始化配置方案 & 软件分享

07 Jun, 2024

系统偏好设置

触控板

  • 开启「轻点来点按」
    • 系统偏好设置 → 触控板 → 光标与点按 → 轻点来点按
  • 开启「使用触控板进行拖移」
    • 系统偏好设置 → 辅助功能 → 指针控制 → 触控板选项 → 使用触控板进行拖移
  • 关闭「轻扫切换页面」
    • 系统偏好设置 → 触控板 → 更多手势 → 轻扫切换页面

Link: https://www.raycast.com/

Install:
